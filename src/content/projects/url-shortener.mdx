# Description

A simple URL shortener built with NextJs and Cloudflare Workers. It uses Cloudflare KV to store the data. It has no ADS and is completely free to use. Totally anonymous and no tracking. The link exiration is set to 24 hours. And it is deployed on Cloudflare Workers.

It gives you a 4 character long URL (Ex. https://jjus.dev/abcd). It is not customizable. Also given is a QR code of the URL for easy sharing.
