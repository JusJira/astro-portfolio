# Description

A dashboard to fetch data from InfluxDB, which is a Database that is used to collect data from IoT sensors such as the ESP8266 that I am using in this project. If you are interested in the hardware side check out [this page](/projects/iot-esp8266/) where I publish the code that I used and the wiring diagram.
